# InjectorsStandPrototype
Розроблено емуляцію роботи стенду промивки форсунок двигуна на базі мікроконтроллера STM32. Використовувались STM32CubeMX та Keil uVision. 
У цілях емуляції замість форсунок були використовані світлодіоди. Частота та скважність регулюється через відповідні енкодери.
